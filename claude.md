# Project Instructions

## Tech Stack
- Frontend: HTML, CSS, JavaScript
- Backend: Node.js + Express
- Database: SQLjs 
- Deployment: Docker Compose

## Coding Rules
- Use clean folder structure
- Write modular code
- Use async/await
- Add comments for complex logic
- Mobile-first responsive design

## UI Style
- Modern UI
- Smooth animations
- Rounded cards
- Minimal aesthetic

## Backend Rules
- REST API architecture
- Use environment variables
- Proper error handling
- Separate routes/controllers/services

## Database
- Use SQLjs 
- Create migration-ready schema

## Docker
- Create Dockerfile
- Add docker-compose.yml
- Production-ready setup

## Important
- Never break existing functionality
- Always explain major file changes
- Don't create any documentation file. 
You are running in Git Bash / bash environment on Windows.
Use ONLY bash-compatible commands.
Do NOT use PowerShell commands like:
- Get-ChildItem
- Select-Object
- $null
- dir
Use:
- ls
- find
- mkdir -p
- grep
- cat
instead.
